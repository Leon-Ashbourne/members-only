# Membory only
A simple members-only project. Only allowed the privilaged can move freely. 

## Prerequisites
The following instructions are followed keeping in view that user already installed **node** of atleast **v22** and **npm v11**. As for database we used psql of minimum **v14** .
You are advised to include your own **.env** file appropriate to your local environment.

## How to use
1. Build the application by running `npm install`.
2. Then run `node app.js` to start the application.
